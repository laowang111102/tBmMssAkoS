# 旅游景点推荐系统 python5

## 项目概述

旅游景点推荐系统是一个基于Python语言的Web应用，结合Vue.js前端框架和Mysql数据库，采用Django框架进行开发。该系统旨在为用户提供精准的旅游推荐服务，同时为管理员提供便捷的管理界面。

## 技术栈

- 后端：Python, Django
- 前端：Vue.js
- 数据库：Mysql

## 功能模块

### 用户功能

- 首页浏览
- 首页推荐内容展示
- 在线反馈
- 旅游资讯查询
- 景点信息展示
- 景点门票查询

### 管理员功能

- 登录认证
- 后台首页
- 在线反馈管理
- 旅游资讯管理
- 景点信息管理
- 景点类型管理
- 景点门票管理
- 用户管理
- 轮播图管理

## 系统角色

- 普通用户：浏览和获取推荐信息，提交反馈等。
- 管理员：系统维护者，具备所有管理权限。

## 运行环境

- Python 3.x
- Vue.js 2.x
- Mysql 5.x
- Django 2.x

## 部署步骤

1. 准备Python运行环境及相应依赖库。
2. 安装Mysql数据库，并进行配置。
3. 部署前端Vue.js项目。
4. 启动Django后端服务。

## 目录结构

```
根目录/
├── backend/       # 后端代码目录
│   ├── apps/      # Django应用目录
│   └── manage.py

    ├── node_modules/
    ├── src/
    └── ...
```
